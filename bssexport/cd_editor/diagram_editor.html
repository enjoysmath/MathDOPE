About Diagram Editor

MathDOPE's Diagram Editor is an adaption of Varkor's original quiver; it's a responsive GUI designed by Dan Donnelly and Peter Shi in December 2023, using BootStrap Studio and HTML/CSS/JS. 

MathDOPE's Diagram Editor is part of the bigger project MathDOPE, which includes storing diagrams in a Neo4j database from a custom Django backend. Source code for the entire MathDOPE project, including the Diagram Editor, can be found on GitHub.
